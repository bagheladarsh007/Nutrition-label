# Nutrition-label
## Description
This project provides the HTML and CSS code for creating a nutrition label for displaying nutritional information.
 You can use this code to easily integrate a nutrition label into your website or web application. The label is designed to be visually appealing and informative.

##  Project Structure
The project consists of two main files:

`index.html:` This HTML file contains the structure and content of the nutrition label.

`style.css:` This CSS file provides the styling for the nutrition label.

## HTML Structure
The index.html file contains the following HTML structure:<br>

`<header>:` Contains the title, servings per container, serving size, and a divider.<br>

`<div class="calories-info">:` Displays the amount of calories per serving.<br>

`<div class="daily-value small-text">:` Lists the daily values of various nutrients.<br>

`<p class="note">:` Provides a note about the daily values.<br>

## CSS Styles
The `style.css` file contains CSS styles for formatting and styling the nutrition label. 
It includes styles for typography, layout, and visual elements such as borders and dividers

Key CSS classes and their descriptions:

`.label:` Styles the overall label container.

`.divider:` Adds a horizontal divider line.

`.bold:` Applies bold font weight to text.

`.large, .medium:` Styles for larger and medium-sized dividers.

`.small-text:` Reduces the font size for smaller text.

`.calories-info:` Styles the container for displaying calories.

`.right:` Aligns text to the right.

`.indent, .double-indent:` Adds indentation to text.

`.daily-value:` Styles for the daily value section.

`.note:` Styles the note section at the bottom.
